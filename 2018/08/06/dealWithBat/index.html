C++调用批处理

批处理（Batch）也称为批处理脚本，即对某对象进行批量的处理，通常被认为是一种简化的脚本语言，应用于DOS和Windows系统中，由DOS或者Windows系统内嵌的命令解释器（CMD.EXE）解释运行，扩展名为：bat。

常见的批处理包括DOS批处理和PS批处理，PS批处理基于Photoshop,用来批量处理图片的脚本；DOS批处理基于DOS命令，用来自动地批量执行DOS命令以实现特定操作的脚本。批处理文件是由一条条DOS命令组成的普通文本文件，可以用记事本直接编辑或用DOS命令创建，也可以用DOS下的文本编辑器Edit.exe来编辑。

举例说明：

创建并编写a.bat，来打开filezilla.exe。然后利用C++编写程序，来调用a.bat，从而执行filezilla.exe。具体过程如下：

  1. 创建a.bat（文本编辑器就可以创建和编辑，本文用的Notepad），并编写a.bat，执行的操作为打开filezilla.exe。

  1. 编写C++程序来调用a.bat，有三种方式：

（1） 创建进程来执行a.bat，头文件为#include <Windows.h>。

PROCESS_INFORMATION pi;
STARTUPINFO si;
ZeroMemory(&si, sizeof(si));
si.cb = sizeof(si);
si.hStdInput = GetStdHandle(STD_INPUT_HANDLE);
if (CreateProcess(L"E:\\svn\\a.bat",
    NULL,
    NULL,
    NULL,
    FALSE,
    CREATE_NO_WINDOW,
    NULL,
    NULL,
    &si, &pi))
{
    WaitForSingleObject(pi.hProcess, INFINITE);  //等待bat执行结束
    CloseHandle(pi.hProcess);
    CloseHandle(pi.hThread);
}

（2） 利用头文件#include <Windows.h>中的WinExec函数。

//方法二
WinExec("E:\\svn\\a.bat",SW_HIDE);

（3） 利用C标准库中的system函数，头文件#include <stdio.h>和<stdlib.h>。

//方法三
system("E:\\svn\\a.bat");

注：bat文件路径要用双斜杠，单斜杠不对，因为C++中会将第一个斜杠当作转义字符。
